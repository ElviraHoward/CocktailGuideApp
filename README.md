# CocktailGuideApp

CocktailGuideApp - it's a pocket guide of cocktails for home cooking, developed in a clean architecture way, based on free [API](https://www.thecocktaildb.com) and using:

- Room
- Retrofit
- Coroutines
- Navigation Components
- Dagger 2
- View binding
